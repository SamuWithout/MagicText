En la carpeta GeneradorDocumentos se encuentran los archivos que forman la app, GUI.py y pdfGenerator.py ambos con sus respectivas lógicas

Para ejecutar la app solo se debe ingresar en la carpeta desde la consola y colocar el comando python gui.py.
La app cuenta con un menú principal con dos botones, el botón de Documento APA es el que te permite crear el archivo pdf con apa y evidentemente el botón de salir es para cerrar la app.
En la pantalla del documento apa puedes elegir para crear la portada y el cuerpo del documento en cuestión con los diferentes campos de textos para rellenar (Es imporante aclarar que esto se encuentra en una etapa temprana de desarrollo y es muy basico)
